# Test Operators

Operators used with `test`, `[`, and `[[` for conditional expressions.

## Categories
- [File Tests](file-tests.md) - `-f`, `-d`, `-e`, etc.
- [String Tests](string-tests.md) - `-z`, `-n`, `=`, `!=`
- [Numeric Tests](numeric-tests.md) - `-eq`, `-ne`, `-lt`, `-gt`
- [Logical Operators](logical-operators.md) - `!`, `-a`, `-o`
