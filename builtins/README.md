# Shell Builtins

Built-in commands that are part of the shell itself.

## Categories
- Command Execution (`command`, `exec`, `eval`)
- Job Control (`bg`, `fg`, `jobs`, `kill`)
- Variables (`export`, `unset`, `readonly`, `set`)
- Navigation (`cd`, `pwd`)
- I/O (`read`, `echo`, `printf`)
